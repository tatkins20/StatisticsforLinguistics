10

There was a significant main effect of Proficiency on Writing Score, F(1, 1) = 5.2544, p <.001. This effect was medium, p2 = 0.066. There was also a significant main effect of Subtitles on Writing Score, F(1, 1) = 3.6962, p = 0.05963. This effect was small, p2 = 0.043.

There was a significant interaction effect between Proficiency and Subtitles, F(1, 1) = 64.9615, p <.001}. This effect was large, p2 = 0.516. This interaction showed that it is the most significant effect since partial p2 assesses the effect sizes of each effect while partialling out the other effects. It also shows that a combination of proficiency and subtitles shows more significant correlations of effect on writing score. Specifically, the writing score was higher for High Proficiency (M = 9.03, SD = 5.88) than for Low Proficiency (M = 6.97, SD = 4.23) in L1 Subtitles, but they differed in L2 Subtitles with higher scores for L2 Subtitles (M = 8.87, SD = 5.95) than for L1 Subtitles (M = 7.13, SD = 4.21) in L1 Subtitles.
